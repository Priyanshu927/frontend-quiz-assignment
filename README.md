# Test Your Knowledge – Quiz UI

A React + TypeScript + Tailwind CSS implementation of the **Test Your Knowledge** quiz for the Frontend Intern assignment.

## Tech Stack

- React + TypeScript (Vite)
- Tailwind CSS
- npm

## Features

- Multi-step quiz with 15 questions (analytical, reasoning, and technical).
- Intro screen with name and register number.
- Animated “Best of Luck!” paw sticker.
- Progress line that moves with each question.
- Prev / Next navigation and result screen with percentage score.
